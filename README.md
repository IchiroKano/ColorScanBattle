# 子どもフェスタ2025 の体験コーナー［仮称：AIで召喚バトル］


## １．（使い方）プレイヤー召喚編

### 起動
$ python video_lego_full.py

### 操作
[ESC]終了 [Q]規準色を記憶 [W]プレイヤー１出力 [E]プレイヤー２出力

１．カメラウィンドウが表示されたら、最初の基準となるカラーバランスを取得する。
２．プレイヤー１のカラフルな物探をカメラにかざす。[w]キーを押す
３．プレイヤー１のパラメータが生成される。
４．同様に、プレイヤー２のパラメータを[E]キーで生成する。
５．[ESC]キーを押して終了する。

## ２．（使い方）プレイヤー対戦

### 起動
$ python game_main.py

### 操作
１．［リロード１］…　プレイヤー１を召喚する
２．［リロード２］…　プレイヤー２を召喚する
３．［対戦スタート］…　バトル開始
４．対戦がスタートして、６ターンもしくはHPがゼロになるまで戦う
